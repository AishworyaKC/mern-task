# Question 02 – Multi Step Form

A simple multi-step form built using React and Tailwind CSS with controlled inputs and basic validation.

##  Features

* Multi-step navigation (Next / Back)
* Controlled form inputs
* Step-based validation
* Clean UI with Tailwind CSS

##  Tech Stack

* React (Vite)
* Tailwind CSS

##  How to Run
npm install
npm run dev

## 📌 Description

This form contains 3 steps:

1. Name input
2. Email input (with validation)
3. Password input (min 6 characters)
